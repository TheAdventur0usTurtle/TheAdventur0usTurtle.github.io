This Goofy Little Website of Mine

This is my totally super awesome project for the first semester of my computer programming 1/2 class. I've been addign quite a bit to it outside of class though, so it's become more of a personal project for me now. I've always wanted to make my own website, I never knew exactly where to start with it. Learning the basics from scratch has been incredibly helpful. In fact, "My Completely Arbitrary Thoughts" at the top of this page on my website contains my first thoughts on HTML, where I was writing down anything that came to mind because I needed to write something to meet the class requirenment.

The layout may look a bit wonky right now, but it will make more sense once I add more to the website. Eventually, I would like to have a little orange car with a pen attatched to the back of it to drive around and draw on it. In the more distant future, I'd like to create another website from scratch for posting (and potentially selling) my projects online. More specifically, doing small costume part comissions. I still would need to learn how to set up a payment system that works and is secure, but I can definitely make the website look visually appealing and functional with the skills that I currently have.

If you are a particularly techy person, you may be wondering how I created this weird staggering effect. It was actually pretty easy. I broke down the webite into three distinct columns in my mind, so its easier think about. Then, I set a maximum width for every element, with headings being slightly wider than the body text to make it visually appealing. After that, I created three classes, one for each column. In CSS, I aligned them using "align-items". However, since this only applies to the children of an element, I have to wrap each element in a div and THEN assign a class to it. Other than that, that's pretty much it.
